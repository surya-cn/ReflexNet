How to get a Groq API Key

  1. Go to the Groq Console and log in or sign up.
  2. Navigate to the API Keys section in the sidebar.
  3. Click Create API Key.
  4. Give the key a name (e.g., "ReflexNet") and generate it.
  5. Copy the key immediately (it won't be shown again) and paste it into the setup screen.
